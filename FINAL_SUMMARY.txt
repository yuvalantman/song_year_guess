QUICK START (5 STEPS, 5 MINUTES):

  1. Create Spotify Developer App
     → https://developer.spotify.com/dashboard
     → Create App, copy Client ID

  2. Setup Local Environment
     → cp .env.example .env.local
     → Add your Client ID to .env.local

  3. Add Redirect URI
     → Spotify Dashboard → Edit Settings
     → Add: http://localhost:3000/callback

  4. Install Dependencies
     → npm install

  5. Run Local Server
     → npm run dev
     → Opens http://localhost:3000

════════════════════════════════════════════════════════════════════════════

DEPLOY TO PRODUCTION (5 MINUTES):

  1. Push to GitHub
     git init && git add . && git commit -m "Initial"
     git push -u origin main

  2. Create Vercel Project
     → vercel.com → New Project → Import from GitHub

  3. Add Environment Variables
     → VITE_SPOTIFY_CLIENT_ID = your Client ID
     → VITE_SPOTIFY_REDIRECT_URI = https://your-project.vercel.app/callback

  4. Update Spotify Redirect URI
     → Spotify Dashboard → Edit Settings
     → Add: https://your-project.vercel.app/callback

  5. Deploy
     → Click Deploy on Vercel

  Live at: https://your-project.vercel.app
